👥 People 人事部
招人 → 入职 → 排班 → 纪律 → 离职 · 5家店全部
80/20 关键：80%的问题来自"缺人"和"招错人"。把招人预算和入职流程管好，其他事少一半。
主要工作流程（按顺序）
  1. 招人 — GM在店里频道说"need 2 server" → Hermes查预算 → 不够就停 → 够就开招聘帖
  2. 筛人 — Hermes打分排名 → GM挑人面试 → 面试完在Slack说"hire" → 写Brain
  3. 入职 — Hermes生成I-9/7shifts账号/PIN → GM带新人填表 → 填完拍照发Slack → Hermes记Brain
  4. 排班 — GM在7shifts排班（自己店）→ 系统发排班给员工 → Hermes看人手够不够 → 不够7AM报你
  5. 日常管理 — GM管出勤/纪律 → 小事coach（先教）→ 重复犯错写警告（要纸质+拍照发Slack）
  6. 离职 — floor staff离职 → Coach Sally批 → manager离职 → Louis批 → Hermes做离职文件
规章制度
规矩1：招人先查预算。超预算 = 停，不商量。
规矩2：入职必须填I-9 + 7shifts账号 + PIN。缺一样 = 没入职完。
规矩3：纪律先教后罚。第一次coach（口头教），第二次写警告（纸质+拍照），严重的事直接写警告。
规矩4：开除floor staff → Coach Sally可以决定。开除manager → Louis决定。永远不自动开除。
规矩5：GM只看自己店。查别的店 = 拒绝。

  3. 不推倒重来，改坏的就行
  4. 一个session做一件事，做完关掉
  5. 先有roadmap再动手
  6. Desktop = 做东西，Slack = 管公司，别混
  7. 查数据，别猜
  8. 一次改一个东西，测好再下一个
  9. Brain是真相，文档会骗人
  10. 做完就走，别反复看
