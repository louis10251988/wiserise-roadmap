Add a new skill加新技能
  1. Create创建 ~/.hermes/skills/<domain>/<name>/SKILL.md
  2. Frontmatter:文件头： name · description · permission: louis-only|team-safe
  3. If louis-only → add to若 louis-only → 加入 config.yaml skills.platform_disabled.slack
  4. Critical agent skill →关键 agent 技能 → hermes curator pin <name>
  5. Restart gateway if team config changed改了团队配置就重启 gateway
  6. Rebuild this Dojo:重建本道场： rebuild skill dojo
